# vitadock
charging dock app for playstation vita

Written with the help of Gemini 3 Pro
Features:
Has a clock
shows battery percentage
shifts pixels to prevent burn in (should work:/)

Installation:
Firstly install registry editor > CONFIG/POWER_SAVING/suspend_interval > set this to 0
Thanks [U/Paddel06](https://www.reddit.com/r/VitaPiracy/comments/1b2pfmg/comment/ksna58i/?utm_source=share&utm_medium=web3x&utm_name=web3xcss&utm_term=1&utm_content=share_button)
Then transfer the OLED Dock vpk to your vita via PC using vitashell and install the vpk in vita.
